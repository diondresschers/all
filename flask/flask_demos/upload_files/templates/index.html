Handling File Uploads With Flask - miguelgrinberg.com

Ignore the message 'No file chose' after uploading, as the page is reloaded and defaulted. Check if the file actually is uploaded

You can make sure the directory has been created that has been defined in the `app.config['UPLOAD_PATH']`, which in this case is: 'static/uploads'`. Otherwise it will be created but raises initially an error.

See all uploaded images.

If you use WTF-forms:

from flask_wtf import FlaskForm
from flask_wtf.file import FileField
from wtforms import SubmitField

class MyForm(FlaskForm):
    file = FileField('File')
    submit = SubmitField('Submit')
